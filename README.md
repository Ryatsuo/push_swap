# 🔢 push_swap

## Description

**push_swap** est un projet d’algorithmie dont l’objectif est de trier une pile d’entiers en utilisant un nombre limité d’instructions et deux piles (`a` et `b`).

Le programme doit afficher sur la sortie standard la liste des instructions nécessaires pour trier la pile, en respectant strictement les règles imposées par le sujet de l’école 42.

---

## 🎯 Objectifs du projet

- Comprendre et manipuler des **structures de données**
- Implémenter un algorithme de tri optimisé
- Minimiser le nombre d’instructions
- Gérer les erreurs et les cas limites
- Respecter la **norme 42**

---

## 🧠 Règles du jeu

- Deux piles : `a` et `b`
- Au départ, `a` contient tous les nombres, `b` est vide
- Le but est de trier `a` dans l’ordre croissant

### Instructions autorisées

- `sa`, `sb`, `ss`
- `pa`, `pb`
- `ra`, `rb`, `rr`
- `rra`, `rrb`, `rrr`
